About my website

My website currently only contains a blog, in which I give advice on software, hardware and more tech topics. Did you know? I built this whole website pretty much from scratch. No stolen templates, no website builder, no third party CSS framework.

I started re-programming my whole blog in February of 2022. The website is still under active developement and I'm constantly testing new features to further improve the user experience. Because of that, there might be some slight problems here and there, so please report issueson my GitHub! Thanks.
